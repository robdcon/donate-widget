Method 1 Basic IFrame Embed (Recommended)

The simplest way to embed the widget. Works on any website, CMS, or platform.

HTML Code:

<!-- Macmillan Donate Widget -->
<iframe 
  src="http://localhost:3001/embed"
  width="100%"
  height="800"
  frameborder="0"
  scrolling="no"
  title="Macmillan Donation Widget"
  style="max-width: 500px; margin: 0 auto; display: block;"
></iframe>

üîß Troubleshooting

Common Issues:

  • Widget not loading: Check that the iframe URL is correct
  • Height issues: Use the auto-resize script or set a fixed height of 800px
  • Stripe errors: Ensure Stripe key is configured (test mode for POC)
  • HTTPS required: Stripe requires HTTPS in production
  • Cookie issues: Some browsers block 3rd party cookies in iframes
