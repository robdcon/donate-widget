Example 1: Basic Inline Widget

What you're seeing: The donation widget is injected into the #donate-widget element below using the inline section script.

GTM Code Used:

<script>
  window.MACMILLAN_DONATE_SECTION_CONFIG = {
    selector: '#donate-widget',
    wrapper: true,
    autoResize: true
  };
</script>
<script src="https://donate-widget-7rbvheu4a-robdcons-projects.vercel.app/donate-section.js"></script>

Page HTML:

<div id="donate-widget"></div>

How It Works

  1. Script loads when page loads
  2. Finds the element with selector #donate-widget
  3. Creates an iframe with the widget
  4. Injects it into the target element
  5. Auto-resizes based on widget height
Open browser console (F12) to see:
  • "Macmillan Donate Section initialized"
  • GTM event: donate_widget_injected
  • Configuration object

Console Commands

Try these in your browser console (F12):

// Check if loaded
console.log(window.MacmillanDonateSection);

// View configuration
console.log(window.MacmillanDonateSection.config);

// Remove widget
window.MacmillanDonateSection.remove();

// Re-inject widget
window.MacmillanDonateSection.inject();

// Update config and reload
window.MacmillanDonateSection.updateConfig({
  amount: 100,
  type: 'one-time'
});
